3. breadcrumbs
Forma de usar la barra reutilizable en otros templates
En las vistas se pasa como contexto las urls necesarias
Ejemplo:
breadcrumb_list = [
    {'label': 'Inicio', 'url': '/'},
    {'label': 'Usuarios', 'url': '/usuarios/'},
    {'label': 'Detalle', 'url': ''},  # El último no necesita URL
]

Y en el Html en el que se vaya a usar
{% include "components/breadcrumbs.html" with breadcrumbs=breadcrumb_list %}

4. Loading Component
Forma de usar el componente de carga simplificado en cualquier parte de la aplicación

a. Para formularios automáticos (el loading aparece al enviar el formulario):
<form method="post" action="/ruta/" data-loading="true">
    {% csrf_token %}
    <!-- Contenido del formulario -->
    <button type="submit">Enviar</button>
</form>

Atributos disponibles:
- data-loading="true" - Activa el loading automático
- data-loading-class="nombre-clase" - Añade una clase personalizada al loading
- data-loading-theme="dark" - Aplica un tema (opciones: "dark", "mini")

b. Usando JavaScript:
<script>
    // Mostrar loading
    loading.show();
    
    // Mostrar loading con tema oscuro
    loading.show({ theme: 'dark' });
    
    // Ocultar loading
    loading.hide();
</script>

c. Loading en contenedor específico:
<script>
    // Mostrar loading en un div específico
    const loadingId = loading.showInContainer('#mi-contenedor');
    
    // Ocultar loading
    loading.hideFromContainer(loadingId);
</script>
